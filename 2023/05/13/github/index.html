部落格前身

早在2年多前（2021年），我就已經開始以 「Study With Jeremy | 傑洛米的學習手札」這個品牌經營我的部落格。最剛開始，我在 Pixnet痞客邦這個免費部落格平台撰寫文章。但過沒多久，卻發現痞客邦上的閱讀體驗十分差勁，充滿了難以控制的廣告。我之後不斷嘗試了許多不同平台，如 Wordpress、Medium、Vocus等等，最後決定要自己購買主機，購買網域名稱（studywithjeremy.com），以自架網站的方式，搭配 Wordpress 建構我的網站。但主機和網域都是訂閱制的，每年都會有一定的開銷，對於我而言，這筆開銷其實是可有可無，而且不論費用多寡，只要知道自己每年都需要付錢，文章寫起來就充滿壓力。
這個品牌在自架網站上經營了約3年後，近期打算再開設一個資工系專屬的部落格網誌，又剛好在資工系的專題中有用到 Github 服務，得知每一個 Github 帳號可以以 github.io 的網域價一個網站，因此我決定開始使用 Github上 的服務。

為什麼選擇 Github？

綜觀網路上各種部落格平台比較的文章與分析，基本上沒有人會把 Github 作為考量的選項之一。
其中的原因不外乎是因為 Github 充其量只是一個讓你放程式，並且能夠 Host你的程式的主機，其餘的程式內容還是需要使用者自己設計，這對於一般沒有程式經驗的人有著較高的技術門檻。
但其實在 Github 上架網站的好處滿多的，其中最吸引我的是

主機完全免費呀！

你沒有聽錯，真的是完全免費！
另外，用過Wordpress 的讀者就會發現，雖然平台提供很多部落格主題可以套用，但這些主題不是沒有可以讓你完全自訂部落格外觀的選項，不然就是要付費額外解鎖更多自訂功能。對於愛好自由的我，這樣綁手綁腳的怎麼會是可以接受的呢？在 Github 上，你需要做的，就是找到一個你喜歡的主題，將程式碼放上去。想改哪邊就改哪邊，自由度超高。當然，前提是你要看得懂這些程式碼。
以列點的方式說明，在 Github 上架網站有以下的好處：

  1. 完全免費。就像前面說過的，主機費用為0，網域費用為0（如果你使用預設的 username.github.io），流量和上線時間也都沒有限制。也不會有原生的廣告。就是完全免費、免費、免費！
  2. 安全性與隱私高。通常在 Wordpress上架網站，我們都會額外下載安全性相關的 plugin，也會時不時就備份一下資料。Github 是微軟的服務之一，而微軟身為全世界最大的網路服務供應商，安全與上線時間基本上都是最佳的。
  3. 可以直接套用主題。這算是優點嗎？若真要比主題， Wordpress的主題更多、更豐富，但 Github 的主題絕對才是自訂性最高的一個。哪邊的字體要變大、哪邊的圖片要置中（css的置中真的是一門學問…）等等的設定，只要你找的到，都可以改成自己理想中的樣子。
  4. 自訂性高。同上點所述，想改什麼都可以改，但這點指的自訂性，是指說整個網頁的架構和網站上的所有檔案，你都有權限可以修改。自訂性高到一個極致！
  5. 一個好的學習環境。好的學習，首先需要有學習動機。而當你的 Github 部落格顯示錯誤，或出現任何問題，你會你力去排除問題。ㄟ，正好，學習動機有了，相信在架設部落格的過程中，每個人都可以學到更多知識與技能！
    然而，在 Github 上架網站有著眾多好處，當然也會有它的短處。

在 Github 上架網站的缺點

  1. 只能架設靜態網站。缺點之一，就是你只能架靜態網站。在 Wordpress上，你可以增加各種動態的 WP外掛，但在 Github 上的網站，只能是預先由HTML生成的內容。其實這也不算是種限制，就是呈現的方式不同。但如果足夠有信心，在Github 上的網站看起來絕對不比 WP上的遜色。
  2. 樣樣自己來。Github 僅提供網站託管平台。網站裡頭的程式碼、內容等等的你都得自己生成，看是要套用 theme也好，自己純手工雕也好，但總是需要你自己來完成。Theme的更新等等的，也得由你手動完成它（這時候就別指望 Github Copilot了）。
  3. SEO，靠自己。談到SEO，這時候你就會懷念 WP上的一堆SEO Plugin，簡單點一點，下載安裝，你的網頁流量就會增加。但在 Github 上沒這種好事，文章標題、分類、標籤、縮圖連結等等的都需要自己來做。以下也提供我的文章都會做的 SEO。

Github適合那些人

說了這麼多，來總結一下 Github適合哪些人用來架部落格吧。

  1. 對於程式語言有基礎的人。畢竟還是要寫程式，如果沒有寫過程式，或是不想花時間寫程式的話，那建議可以用最主流的 Wordpress就好。但如果你對於寫程式滿腔熱忱，或是想試看看用程式寫出一個完整的網頁，那Github就非常適合你。
  2. 不想花太多錢的人。Github上架設網站完全0費用，因此如果你只想把部落格當興趣，偶爾寫寫，不想產生額外費用的話，那可以試試看 Github。
  3. 不怕麻煩，熱衷於學習的人。資工這條路，能走多遠、走多廣全看你的學習態度與學習量。而在 Github上架網站的這個過程中，你將會有許多機會學習。如果你不怕麻煩，那不妨可以試試看。
    有任何問題的話，都可以在留言處留言，我會盡力幫助你~
