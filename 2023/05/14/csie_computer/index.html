前言

讀資工系到底要選擇怎樣的電腦呢？

相信這是每個就讀資工系的學生在剛入學時一定會有的疑問。
是不是心裡想著要買那種7、8萬，配備 4070獨顯、16吋 2K 大螢幕、RGB鍵盤的電競筆電？
是不是覺得自己會像電影裡的駭客一樣，在暗暗的房間裡面看著三螢幕，螢幕上自動跑著一堆綠色的 code
但現實上，大多數時間你只會在一台筆電前凝視著螢幕，從程式海中找到自己少打的分號。

對，說實話，要應付大學的課業，你不需要太好的電腦。

便宜的電腦就足以應付你的課業

資工系大一、大二時通常會開始學習一些基礎的語法，使用到的語言可能包含C、 C ++、 Java、 C#等等。
但基本上學生不會參與到太大的專案，因此需要的電腦配置要求也就不會太高。
還記得在大二時修過一堂網際網路資料庫程式設計的課程，課程的期末作業是要完成一個全端網頁，包含前端網頁的呈現（HTML + CSS + JS + PHP）、後端的後臺管理與資料庫（MySQL + PHP），雖然看起來很多很複雜，但其實這些程式在一般的輕薄型筆電就可以做到。
最基本的筆電配置建議如下：

  • CPU：建議能選到 Core i5以上，建議直上 i7。因為會大量寫程式，因此處理器建議選擇高級一點的，程式跑起來比較穩。
  • RAM：12GB 以上。有些程式和軟體運行時很吃記憶體，尤其是跑模擬器的時候。
  • 硬碟：建議 512GB以上 SSD。
  • 顯卡：獨顯不是必備的，現在的內顯效能也有一定水準。獨顯的功用彰顯在進行機器學習和人工智慧相關作業的時候。我之前有一堂人工智慧課程，因為筆電只有內顯，要訓練 model 只能用 Google Colab。
  • 連接阜：建議買還配備有 USB Type A 的電腦，不然藥用個 USB還得另外帶轉接頭。另外可以選擇有 VGA 接頭的筆電，在報告或演示時要投影到螢幕上比較方便。現在的電腦基本上都會配備至少一個 USB Type C，建議可以另外買一個 Type C Hub，擴充你的連接阜。
  • 螢幕尺寸：建議選擇13吋以上的螢幕，1080 FHD 以上。
  • 重量：建議選擇 2公斤以下的電腦，攜帶比較方便。
  • 額外提供一個小建議，建議選擇充電器是 Type C 的筆電。一來，不指定充電器表示之後如果需要額外購買充電器，就不一定要買原廠貴到爆的充電器了，只要瓦數夠都可以衝。二來，如果忘記帶充電器出門，可以跟同學借一下來用的機率比較大。
  • 再另一個提醒，建議不要買 Apple 的 Macbook，否則你很有機率遇到學校上課用的 Compiler 沒有 Mac版本，或老師教的指令還需要自己額外上網查 Mac 版本的窘境。

筆者使用的筆電

我自己使用的筆電視在大一上剛開學時買的。

我用的是 Samsung Galaxybook Flex 15.6吋的。
什麼？三星有出電腦嗎？我常常被這樣問。
答案是有的，只是台灣沒有進。
以下是這台電腦的硬體資訊。

  • CPU：Intel Core i7-1065G7
  • RAM：12GB LPDDR4x
  • 儲存空間：512GB NVMe SSD
  • 顯示卡：Intel Iris Plus
  • 螢幕： 15.6吋 1080p QLED 觸控顯示螢幕
  • 連接阜：USB-C x 3（兩個支援 Thunderbolt 3）、3.5mm 耳機插孔、UFS / microSD 讀卡器
  • 重量：1.58 公斤
  • 售價：1399 美金

我當初會選擇這台筆電，特地從美國運回來的原因有以下幾點：

  1. 螢幕大且是 QLED 螢幕，長時間打程式還是可以賞心悅目。
  2. 連接三星生態系，資料同步很方便。
  3. 非常輕薄。三星工藝真的好得沒話說，這台電腦真的很輕，攜帶超方便。
  4. 質感超級好。這是同學公認的～第一次看見我的電腦的同學都會問我這電腦是怎麼買的，那個典雅藍色怎麼那麼漂亮。這台電腦輛也好，全機身金屬，不像一般的電競筆電多是塑膠材質。
  5. 續航好：這台電腦的續行真的不錯，基本上帶去學校一天不太需要充電。
  6. 三星內建軟體好用：筆記軟體、設定等等的，這些三星獨有的軟體很方便。

總結

其實吧，大學資工系對電腦的要求真的不高，主要就是處理器和記憶體不要太差即可。
另外就是選擇一台自己看得順眼，後續維修有保障的電腦和品牌即可。
祝大家寫程式愉快，也祝大家通靈順利～
