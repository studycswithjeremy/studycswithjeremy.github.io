Markdown 簡介

Markdown 是一種輕量級的標記語言，最初由 John Gruber 和 Aaron Swartz 在 2004 年設計，旨在讓人們以易讀易寫的方式撰寫文件。Markdown 不但易於學習和使用，還可以通過簡單的文本編輯器就能生成豐富文本格式的文件。因此，它在撰寫部落格、文件、閱讀筆記等場景中被廣泛應用。本部落格的文章也全是以 Markdown 語法撰寫而成。
但由於筆者是初次接觸 MD（除了有幾次在寫 Github readme 時有稍微用過外），需要時常上網查詢語法，因此想說不如自己就來寫一篇 MD 語法的筆記，用來記錄一些常使用到的 MD 語法。

用 1 ~ 6 個 # 區分 h1 ~ h6， #數量越多標題越小。#與文字間需要一個空格

清單列點

若要產生列點的效果，可以在清單的每個點前加入 *、+、- 符號，即會在每行前面出現小黑點。

  • 第一點
  • 第二點
  • 第三點
    若是需要以數字列點，則直接在每行前面加入數字 + .，如 1. 即可。
  1. 第一點
  2. 第二點
  3. 第三點

Checkbox

Checkbox 語法為 - [ ] uncheck 以及 - [x] check

  • uncheck
  • check

超連結

超連結的語法是 [連結名稱](連結網址 "游標顯示的文字")
